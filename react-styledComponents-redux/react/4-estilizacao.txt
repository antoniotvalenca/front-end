REACT - ESTILIZAÇÃO

O react, por ser voltado a desenvolvimento de user interface, dispõe de bibliotecas de estilização também

1) ÍCONES
  - > npm i react-icons

  - > import { iconeQueQuer }  from 'react-icons/fonteQueQuer'

  -> (... codigo)
    -> <button type="submit">
        <iconeQueQuer />
       </button>

2) CSS
  -> No react, o CSS é feito num arquivo .css genérico, onde, após tudo ser feito, é exportado e, por fim, importado pelo componente que vai receber essa
  estilização ( )
